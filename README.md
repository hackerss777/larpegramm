# ⬡ Larpegramm

Защищённый мессенджер с сине-чёрным стилем.

## 🔒 Безопасность
- Все сообщения шифруются на устройстве (AES-256)
- End-to-end encryption

## 🚀 Запуск
Просто открой `index.html`

## 🧰 Технологии
- HTML/CSS/JS
- Web Crypto API

## 📱 Адаптивность
Работает на телефонах и десктопах.
