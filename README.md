# ⬡ Larpegramm

Защищённый мессенджер с магазином и LarpeCoin.

## 🔒 Безопасность
- Все сообщения шифруются на устройстве (AES-256)
- End-to-end encryption

## 🪙 Магазин
- **Прайм** — 499 LarpeCoin
- Бейджик, скидка 10%, эксклюзивный чат (в разработке)
- Недоделано: темы, стикеры

## 🚀 Запуск
Просто открой `index.html`

## 🧰 Технологии
- HTML/CSS/JS
- Web Crypto API
- LocalStorage (в будущем)
